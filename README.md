Description:

Gym Management System is a modern, full-stack web application built using the MERN stack with React+Vite as the frontend framework for enhanced performance and faster development.
The application leverages the power of Vite, a next-generation frontend tool, to deliver lightning-fast development and build processes, while the backend, powered by Node.js and Express, ensures a reliable and secure API for data management. MongoDB serves as the robust database to store and manage all gym-related data.

Runs the app in the development mode.
Open http://localhost:5173 to view it in your browser.

- for frontend:
  cd frontend/
  npm run dev

- for backend:
   cd Backend/
   node app.js
